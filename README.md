# Trade_Tip（金融助手）
这是一个金融助手，帮助您分析您感兴趣的股票和基金。主要指标是箱体趋势和均线。
This is a financial assistant that helps you analyze stocks and funds that interest you. The main indicators are box trend and moving average.

Rust+Tauri
